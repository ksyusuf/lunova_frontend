# Lunova - Bağımlılıkla Mücadele Platformu

Lunova, bağımlılıkla mücadele eden bireyler için destek sağlayan bir platformdur. Bu proje, Lunova'nın ana web sitesini oluşturur.

## Özellikler
- Kullanıcı dostu arayüz
- Bağımlılıkla mücadele kaynakları
- Blog ve makaleler
- İletişim ve destek

## Kurulum

Projeyi çalıştırmak için aşağıdaki adımları izleyin:

1. Gerekli bağımlılıkları yükleyin:
   ```bash
   npm install
   ```

2. Projeyi başlatın:
   ```bash
   cd lunova-frontend
   npm run dev
   ```

## İkonlar

Proje, Tailwind CSS ve diğer modern web teknolojilerini kullanır. Mini ikonlar için [Heroicons](https://heroicons.com/) kullanılmaktadır.
